# E-Commerce Online Gadgets Shop

## Overview

This is an e-commerce online gadgets shop developed to provide users with a seamless shopping experience for various electronic gadgets. The platform features a modern, responsive design that enhances user interaction, making it easy to browse, compare, and purchase products.

**Note:** The backend functionality is not implemented yet, and the project is currently a work in progress.

## Features

- **User-Friendly Interface:** A modern and intuitive interface that enhances the shopping experience.
- **Product Listings:** Dynamic product listings with filters and sorting options.
- **Product Details Page:** In-depth information about products, including specifications and reviews (backend functionality to be added).
- **Comparison Feature:** Allows users to compare multiple products side by side (backend functionality to be added).
- **Responsive Design:** Optimized for mobile and desktop devices.
- **Dynamic Variations:** Support for different product variations like color, size, and RAM

## Technologies Used

- **Frontend:** React.js, Vite, React Bootstrap
- **Backend:** Planned with Node.js, Express.js (not implemented yet)
- **Database:** Planned with PostgreSQL (not implemented yet)
- **Version Control:** Git, GitHub
